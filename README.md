# AjaxExercises
Some exercises for JQuery, specifically AJAX. Some API usage for generating fake user data and some dog images.

First section:
Button that pulls text from a txt file I made.

Second section:
Button that pulls dog images from a link and displays it, changing the image with each press.

Third section:
Button that generates a fake user with an img and p with an email inside. Both the p and img change with each press.

Forth section:
Button that generates a dog gallery, which pulls 5 dog images at once. When pressed again it replaces already places images.
